# Discord 通道

基于 Discord.js 的通道实现，支持服务器和私信。

## 🔧 配置

```yaml
channels:
  discord:
    enabled: true
    botToken: "MTIz..."
    allowedServers: []
    allowedChannels: []
```

## 🎯 特性支持

- ✅ 文本消息（Markdown 格式）
- ✅ 图片/文件收发
- ✅ 流式编辑
- ✅ 线程支持
- ✅ 反应（emoji）
- ✅ Slash 命令
- ✅ 嵌入消息
